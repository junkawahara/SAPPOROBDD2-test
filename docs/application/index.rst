応用ガイド
==========

このセクションでは、SAPPOROBDD 2.0を用いた実践的な応用例を、
課題指向のガイド形式で紹介します。BDD/ZDDの研究者や実務者が
具体的な問題に取り組む際の参考となることを目的としています。

各ガイドでは、問題の定式化からSAPPOROBDD 2.0による実装までを
ステップバイステップで解説します。基本的なBDD/ZDD操作については
:doc:`../quickstart` および :doc:`../tutorial` を事前にお読みください。
